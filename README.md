# DesignPatterns
This project contains implementation of 22 basic design patterns:
- Abstract Factory
- Adapter
- Bridge
- Builder
- Chain Of Responsibility
- Command
- Composite
- Decorator
- Facade
- Factory
- Flyweight
- Iterator
- Mediator
- Memento
- Observer
- Prototype
- Proxy
- Singleton
- State
- Strategy
- Template Method
- Visitor

There are created many tests and simple **CarApplication** showing context of these implementations.
